# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-XX

### Added
- Initial release
- Portfolio value sensor
- Portfolio purchase value sensor
- Portfolio profit/loss sensor
- Portfolio profit/loss percentage sensor
- Holdings count sensor
- Individual holding sensors for each position
- Configuration flow with account selection
- Support for Easy Equities and Satrix
- Automatic data updates every 5 minutes (configurable)
- Secure credential storage
