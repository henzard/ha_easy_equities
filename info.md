# Easy Equities Integration

Monitor your Easy Equities and Satrix portfolio directly from Home Assistant.

## Features

- Real-time portfolio value tracking
- Individual holding sensors
- Profit/loss calculations
- Automatic updates every 5 minutes
- Support for multiple accounts
- Works with Easy Equities and Satrix

## Installation

Install via HACS or manually by copying the `easy_equities` folder to your `custom_components` directory.
