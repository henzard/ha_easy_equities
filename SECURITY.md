# Security Policy

## Supported Versions

We provide security updates for the following versions:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability, please do **NOT** open a public issue. Instead, please email the maintainers directly or create a private security advisory on GitHub.

Please include:
- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

We will respond within 48 hours and work with you to address the issue.

## Security Best Practices

- Never share your Easy Equities credentials
- Use strong, unique passwords
- Enable 2FA on your Easy Equities account if available
- Keep Home Assistant and this integration updated
- Review your portfolio data regularly
